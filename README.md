# DevBoard

DevBoard is a modern developer dashboard that helps users manage GitHub repositories as projects in one centralized platform. It combines GitHub repository insights with a Kanban-style project tracker, making it easier to organize, prioritize, and monitor project workflows efficiently.
## ✨ Features

- GitHub integration using Personal Access Token  
- Fetch repositories using GitHub API  
- Add repositories as projects to a Kanban board  
- Set priority levels: High, Medium, Low  
- Track project status:
  - To Do  
  - In Progress  
  - In Review  
  - Completed  
- Clean and user-friendly interface  
- Landing page for better onboarding  

---
## ⚙️ How It Works

1. Users visit the landing page and get an overview of the platform.  
2. They enter their GitHub Personal Access Token to securely connect their account.  
3. DevBoard fetches repositories using the GitHub API.  
4. Users select repositories and add them as projects.  
5. Projects are displayed and organized on a Kanban board.  
6. Users can:
   - Assign priorities (High, Medium, Low)  
   - Move tasks across stages (To Do, In Progress, In Review, Completed)  
   - Track progress easily in one place
